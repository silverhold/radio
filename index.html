Custom Radio

designed by Clément Menant and Loïc Goyet for AppVentus

This is some css that allow you to style (in CSS3 and HTML5) the input type=radio elements as a Custom Radio

Dependencies Required

no dependencies required

Available for

CSS LESS

Getting Started

To enjoy it just link to your template or to your stylesheet the css/style.css file.

You can also use the less file available at less/style.less. With this file you can overide easily customize the Less variables listed below in order to define colors, sizes and more.

Bower usage

Install and manage Custom Radio using Bower.

$ bower install custom-radio

Less Variables

The variables are divided in two categories, the modules and the themes. The module variables change the structure and the behave of the component in your template. The theme just change the look without size or layout consequences. Those two kinds of variables are called parameters and you can find them in /less/parameters

  • T @customRadio_transitionDuration: 100ms
  • M @customRadio_transitionDuration: 100ms
  • M @customRadio_height: 20px
  • M @customRadio_width: 20px
  • M @customRadio_borderWidth: 1px
  • T @customRadio_borderColor: #cdcdcd
  • T @customRadio_borderColor-_hover: @customRadio_borderColor
  • T @customRadio_borderColor-_checked: @customRadio_borderColor
  • T @customRadio_borderColor-_hover-_checked: @customRadio_borderColor
  • T @customRadio_bgColor: white
  • T @customRadio_bgColor-_hover: @customRadio_bgColor
  • T @customRadio_bgColor-_checked: @customRadio_bgColor
  • T @customRadio_bgColor-_hover-_checked: @customRadio_bgColor
  • T @customRadio_boxShadow: 0 1px 5px 0 fade(#333, 15%)
  • T @customRadio_boxShadow-_hover: @customRadio_boxShadow
  • T @customRadio_boxShadow-_checked: @customRadio_boxShadow
  • T @customRadio_boxShadow-_hover-_checked: @customRadio_boxShadow
  • T @customRadio_opacity-_disabled: 0.50
  • M @customRadio-label_marginLeft: 15px
  • M @customRadio-witness_height: 8px
  • M @customRadio-witness_width: 8px
  • T @customRadio-witness_bgColor: #e36566
